# MeteoStats
Android app showing current and historical data from weather stations in Tricity.
Application was written in MVVM pattern.

#### Used technologies: 
* Java,
* Android Architecture Components (LiveData, ViewModels), 
* Retrofit 2, 
* WilliamChart, 
* Room, 
* Dagger 2
